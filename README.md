# Tea.ua

### Features
* NextJS

### Host Machine Requirements
* NodeJS 20.17.0
* Yarn 1.22.22

### Installation

1. **Copy project example .env file**
   ```sh
   cp .env.example .env
   ```
2. **Install dependencies**
   ```sh
   yarn install
   ```
3. **Run project**
   ```sh
   yarn dev
   ```
4. **[Go to client view](http://localhost:3000)**
